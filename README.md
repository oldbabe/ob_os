## 开发说明

使用Qt6.2版本打开CMakeList.txt文件后使用 Desktop Qt 6.2.4 MSVC2019 64bit编译即可,初次执行时需要让项目 执行CMake.
编译和执行时需要在根目录提交SDK目录，并添加opencv490运行库,具体位置请查看CMakeList文件。
